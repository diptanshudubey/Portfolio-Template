Intelligent Waste Management System using Image Processing

Objective -The objective of the project is to segregate biodegradable and non biodegradable waste .
Discription -Intelligent Waste Management System project is based on image processing technique.The image processing is done with the help of ML. We use Ultrasonic Sensor,Servo Motor and Arduino UNO Board for hardware parts.Camera is used to detect the object which is placed. Using ML algorithm, the features of the object is detected and stored in a data set. Once the image is been captured by the camera, that features of the image is compared with the database image using ML algorithm find whether there is any similar image and show output accordingly.If object is biodegradable then it will go in biodegradable side of bin and if object is non-biodegradable then it will go in non-biodegradable side of bin.
